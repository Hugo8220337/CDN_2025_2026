\section{Utilização de Inteligência Artificial}
Para o desenvolvimento deste projeto foram utilizadas algumas inteligências artificiais como auxílio ao desenvolvimento do projeto. No entanto a utilização das mesmas não foi utilizada como substituto do trabalho necessário para a aquisição das competências necessárias para o desenvolver, sendo somente utilizadas como recurso para facilitar trabalho repetitivo ou demorado não é relevante para o projeto, tendo sido utilizada para as seguintes tarefas:

\begin{itemize}
	\item Criação do ficheiro CSS e organização das classes do boostrap no frontend
\end{itemize}
